Misfit Grafix

Misfit Grafix is a local Schenectady area custom paint shop that has recently asked me to start a web project to expand their digital footprint beyond Facebook, and Instagram. This is still a work in progress (mainly waiting on the content from the client) but you can see my work so far

reu manager

Following my internship with Notch 8 I was contracted to work on a project originally started by UCSD which aimed to make the process of applying for academic grants more streamlined. I was part of a 7 person team for this project and was mainly responsible for the front end design and build

Thero.org

Thero.org is a San Diego based startup focusing on connecting those in need of mental health treatment with appropriate providers in their local area. I feel very lucky that I was able to be a part of creating something that enriches peoples lives on a daily basis.

Shelter friends

this was my final project at Learn Academy, an app that aims to connect available shelter pets with possible adopters. for this project i was part of a 5 person team, and we were given just under two weeks to plan, and build the app.

Accessibility

This is an oral presentation I gave while at Learn Academy on the importance of accessibility requirements in web projects. I felt that it was an important topic to cover which is often overlooked.
